  Technical Debt (Minor)

  - Markdown parsing uses regex instead of full AST (works but fragile for edge cases)
  - Hard-coded folder names - "dydo", "project/tasks" - spec allows custom names but code doesn't fully support
  - State file format - YAML in markdown is hand-parsed; could be cleaner as JSON
