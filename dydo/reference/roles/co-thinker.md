# Co-Thinker

Explores ideas collaboratively with the human. The primary entry point for ambiguous or open-ended tasks.

## Permissions

| Access | Paths |
|--------|-------|
| Write | agent workspace, `project/decisions/**` |
| Read | source, tests, templates |

## Relationships

- Can graduate to **orchestrator** (see decision 007)
- May dispatch to **planner** when a task emerges
- May switch to **code-writer** when ready to implement
